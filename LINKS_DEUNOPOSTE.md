```html
<table class="resultado-tabela">
    <thead>
        <tr>
            <th>Prêmio</th>
            <th>Milhar</th>
            <th>Bicho (Grupo)</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td class="premio">1º</td>
            <td class="milhar">6470</td>
            <td>Porco (18)</td>
        </tr>
        ...
    </tbody>
</table>
```

## Formato dos Dados Extraídos

Cada resultado é extraído no seguinte formato:

```json
{
    "loteria": "Deu no Poste",
    "horario": "9h",
    "numero": "6470",
    "animal": "Porco",
    "grupo": 18,
    "premio": "1º",
    "timestamp": "2026-01-08T13:00:00",
    "fonte": "deunoposte.com.br"
}
```
